# DiziFilmRadar
A mobile app to find where movies/series are streaming in Turkey.

## Özellikler
- Tek arama ile tüm platformlarda bulunabilirlik
- Kişisel izleme listesi (otomatik güncelleme)
- Platform fiyat listesi
- Günlük veri yenileme
- Yapay zekâ önerileri (V1 içerik tabanlı)

## Durum
Geliştirme aşamasında.
